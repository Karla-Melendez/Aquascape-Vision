# Assets

Directory contains visual assets created during the design and development of Aquascape Vision.

## Structure

Diagrams/ <br />
	System diagrams, information architecture diagrams, workflows, data models, and other visual documentation.

Logos/ <br />
	Project branding assets, including logos, icons, and visual identity explorations.

Screenshots/ <br />
	Screenshots of prototypes, application builds, and user interface iterations.

Sketches/ <br />
	Early hand-drawn concepts, wireframes, and brainstorming materials.
