# Aquascape-Vision

Aquascape Vision is a planned application that helps aquarists design planted aquariums before purchasing plants and hardscape.

## Goals

- Visual tank planning
- Plant knowledge base
- Growth projection
- Future AR visualization

## Current Status

Research and product-definition phase.
